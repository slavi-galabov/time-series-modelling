
This repository contains Jupyter notebooks demonstrating core quantitative modelling techniques used in financial risk analysis and forecasting.

## Included Models
- **ARIMA** – time series forecasting and trend decomposition  
- **ARIMA‑GARCH** – volatility modelling, heteroskedasticity analysis and risk estimation  
- **Monte Carlo Simulation** – stochastic scenario generation using t‑distributed shocks  

## Key Concepts
- Stationarity testing (ADF)
- Model selection (AIC/BIC)
- Volatility clustering and GARCH(1,1)
- Random walk simulation and drift estimation
- Barrier analysis (65% dynamic barrier)
- Forecast visualization and scenario analysis

## Tools
Python, Pandas, NumPy, statsmodels, arch, SciPy, Matplotlib, Plotly, yfinance.
